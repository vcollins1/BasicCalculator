# Basic Calculator App

A responsive basic calculator app built with Blazor WebAssembly as part of a Frontend Mentor challenge.

## Built With

- Blazor WebAssembly
- .NET 10
- Razor components
- CSS

## Design Credit
The UI design was provided by Frontend Mentor as part of one of their frontend development challenges.

- Frontend Mentor challenge: [calculator app](https://www.frontendmentor.io/challenges/calculator-app-9lteq5N29)
